## الأوامر المتاحة 📋

### 🔍 **BIN Information:**
```
/bin [BIN]
```
**مثال:** `/bin 453264`

### 🔧 **Generate Cards:**
```
/gen [BIN]
```
**مثال:** `/gen 453264`

### 🔍 **Check Card:**
```
/check [CC|MM|YYYY|CVV]
```
**مثال:** `/check 4532640527811643|12|2025|123`

### 📝 **Register:**
```
/register
```
